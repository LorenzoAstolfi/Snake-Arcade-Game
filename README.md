# Snake-Arcade-Game
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

## How to play
1. Make sure you have Python 3 installed on your computer.
2. Download the folder `SnakeGame`.
3. Open a terminal or command prompt.
4. Go to the folder where the file is saved.
5. Run the game using this command: `py main.py`.
6. Follow the instructions and type your choices in the terminal.

## Requirements
- Python 3 (if you don't have Python, here's the link: https://www.python.org/downloads)
